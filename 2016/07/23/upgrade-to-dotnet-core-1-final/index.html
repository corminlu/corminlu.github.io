升级到.Net Core 1.0正式版

  • aspnet
  • dev
  • dotnet core

.Net Core 1.0正式版已经发布一段时间了，稳定性又提升了不少，是时候升级了。

卸载旧版本

Windows

卸载RC1

卸载RC2

如果没卸载干净，请到控制面板手动卸载干净

CentOS

直接删除.Net Core目录即可

安装新版本

直接到.Net Core官网下载相应安装包安装即可。

Windows

使用Visual Studio的话需要更新到Update3，然后下载安装.NET Core SDK Installer(Preview 2)。

如果网络比较慢的话，很可能在线安装不会成功，建议先离线下载完毕再安装

1.离线下载

layout方法可能也会下载不下来，可以在失败的时候看看需要哪些文件，然后用下载工具把它们下载到layout下载的目录，可能需要以下文件：

  • DotNetCore.1.0.0-SDK.Preview2-x64.exe
  • DotNetCore.1.0.0-SDK.Preview2-x86.exe
  • aspnetcoremodule_x64_en_rc2_39.msi
  • aspnetcoremodule_x86_en_rc2_39.msi
  • ancm_iis_express_x64_en_rc2_39.msi
  • ancm_iis_express_x86_en_rc2_39.msi
