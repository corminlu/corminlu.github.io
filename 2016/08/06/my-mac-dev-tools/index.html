我的Mac开发工具清单

  • dev
  • mac

记录一下我在Mac下使用的开发工具(部分未使用过)

XCode

这个无需解释

Alcatraz

XCode的插件管理工具，支持XCode 7+

安装

主要使用了以下插件：

  • FuzzyAutocomplete 自动完成比IDE自带好用太多
  • Auto-Importer 自动导入头文件
  • ClangFormat 代码格式化
  • KSImageNamed 自动完成imageNamed:时预览图片
  • OMColorSense 颜色辅助输入UIColor或NSColor实时预览颜色
  • VVDocumenter-Xcode 注释自动完成
  • XcodeBoost 很方便的复制粘贴等
  • CocoaPods 菜单栏直接执行pod命令
  • BBUDebuggerTuckAway 自动隐藏调试区域
  • DerivedData Exterminator 清理XCode 派生数据(DerivedData)
  • InjectPlugin 代码注入支持实时预览
  • RevealPlugin 附加到Reveal中查看视图结构

插件安装在~/Library/Application Support/Developer/Shared/Xcode/Plug-ins/目录下，如不需要时可手工删除或通过Alcatraz删除。

Visual Studio Code

轻量级的跨平台代码编辑器，现在已经有很多插件可用，用来做简单的代码编辑，或者当作markdown编辑器都很不错。

在编写js/ts/es等代码时，可使用jshint/tslint/eslint等代码语法检查。

安装C#(OmniSharp)插件，可编写/调试.Net Core。

BBEdit

收费，也是不错的一款轻量级编辑器

Xamarin Studio

可用于编写传统的.Net Winform/Asp.Net应用，需要安装Mono MDk支持

可通过brew cask安装

JetBrain AppCode

收费，据说写代码的体验比XCode好很多

JetBrain Intellij IDEA

最强大的Java IDE

SourceTree

git/hg源代码管理

可通过brew cask安装

Tower

收费，据说是最好的Git客户端，价格特别贵，实际上和SourceTree差不了多少

CornerStone

收费，Mac下最好的SVN客户端

KDiff3

跨平台的文件对比工具

可通过brew cask安装

设置.gitconfig

Beyond Compare

收费，最好的文件对比工具

Docker

有了它太方便了，另外不要忘记装它的界面管理器Kitematic

Reveal

收费，神器，实时检查、修改、调试iOS/Mac界面

是Mac OS X平台上的一款方便开发者调试IOS应用的开发软件，reveal能够在运行时调试和修改iOS应用程序。Reveal能连接到应用程序，并允许开发者编辑各种用户界面参数，而且会立即反应在程序的UI上

除了使用RevealPlugin来attach以外，还可使用唐巧《iOS开发进阶》的方法，编辑~/.lldbinit

然后在application: didFinishLaunchingWithOptions:方法加入断点，”Add Action”->”Debugger Command”->输入”reveal_load_sim”，并勾选”Automatically continue a er evaluating”。

Chisel

Facebook开源的一款lldb调试工具，虽然没有Reveal方便

Charles

收费，抓包、网络调试工具。感觉没有fiddler好用，可惜fiddler不支持mac

SimPholders

收费，在菜单栏上快速的访问iPhone Simulator中的应用的文件夹

ThemeEngine

提取Assets.car中的图片资源

类似的还有cartool/iOS-Asset-Extractor/acextract。

Unused

找出XCode项目中未使用的图片资源，准确性还可以，但不是100%

Navicat

收费，强大的数据库管理工具

Redis Desktop Manager

Redis GUI管理器

RedisReact

Redis GUI管理器

Sip

有免费版与收费版Pro，屏幕取色工具，你只需要鼠标点一下即可取得屏幕当前位置的颜色值，并将数据自动存到剪切板，方便随时粘贴出来。支持 CSS、UIColor 等多种格式。

ColorSnapper

收费，优秀的屏幕取色工具，除了快速取色外，具有数十种颜色模式，比如可以直接复制为iOS代码的颜色值！

Core Animator

收费，动画制作工具，可导出为objc和swift代码，收费

Framer

收费，移动交互原型设计工具，基于JavaScript，较为诱人的有以下几点：支持手势、动效、状态、基于网页、容易实现、可在任何设备上浏览

Principle

收费，Mac平台交互动效设计神器，五分钟即可制作出一个具有完整交互动画的原型，并且可将交互动画生成视频或者 Gif

xScope

收费，是一款Mac上的设计开发辅助工具，尤其是移动应用的设计，具有10个实用优秀的小工具，如屏幕镜像、屏幕标尺、放大镜、十字定位等等，都是设计时非常实用的辅助工具。

Sketch

收费，轻量易用的矢量设计工具

PaintCode

收费，专业矢量图形绘图工具，通过PaintCode没有编程经验的设计师也能立即画出美丽的控件，图标或其他UI界面元素并直接生成适用于iOS或者OS X的Objective-C代码

CopyClip

1.x免费，2.x收费，专业MAC剪切板管理工具

Paste for mac

收费，剪切板增强工具

Unclutter

收费，剪切板增强，支持放入文件

Noizio

收费，是一款 Mac OS X 下的白噪音应用，可以让自己觉着是坐在一个壁炉前、咖啡馆里、森林里，海边。可以让你忽略其他噪音，专注工作。

Tickeys

跨平台，是一款能够在打字时模拟机械键盘音效的应用，此外还能模拟冒泡、打字机、剑气、爆裂鼓手的音效

可通过brew cask安装
