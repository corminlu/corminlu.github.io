我的Mac软件工具清单

  • mac

记录一下我在Mac下使用的各种软件工具(部分未使用)

系统工具

Path Finder 超赞Mac资源文件管理工具，自带shell面板，可以设置成Solarized配色

iStat Menus 是Mac上最优秀的系统监控工具，能够在系统菜单栏实时监控CPU、GPU、内存、硬盘、网络、温度、电池以及系统时间等

iStatistica 一款高颜值的系统监控工具和iStat Menus类似

Today-Scripts 可以在通知中心显示CPU/内存/硬盘/电池等使用情况

CCleaner 资深系统垃圾清扫软件

CleanMyMac 强大的mac系统清理工具

Alfred 效率神器，秒杀SpotLight渣渣。有各种Workflow，用起来非常方便

Flashlight SpotLight插件，据说和Alfred功能类似

BetterZip 快速的压缩工具

cDock 是一款专门为OS X系统定制的小插件。可以帮助我们自定义dock栏的主题，改变其颜色，背景，透明。

BetterTouchTool 是一款专为Mac用户开发的 Magic Mouse 鼠标 和 TouchPad 的功能增强软件。

DaisyDisk 系统清理工具，他可以帮助你了解磁盘使用状况，并进行准确的分析，让你非常轻松就能揪出占用磁盘空间的那些文件

Disk Sensei 是一款磁盘空间管理工具，以直观的区块、数字和颜色来显示硬盘上文件夹，文件大小，且功能强大的Mac磁盘清理软件。

EuDic 欧路词典是一款Mac上的非常优秀的词典工具，支持屏幕取词、划词搜索、生词本、翻译朗读、词库扩展等等功能

1Password 密码管理工具

Hands Off 是一款在Mac上类似防火墙的工具，它可以帮助你控制程序是否访问网络或者磁盘。并对系统不可信的进程和信息进行监控。

网络工具

Chrome 浏览器，可通过brew cask安装

Postman Chrome必不可少的网络请求插件

Firefox 浏览器，可通过brew cask安装

Firebug Firefox必不可少的调试工具插件

uBlock Origin Chrome/Firefox去广告插件

QrCode 浏览器都需要有生成当前页面地址的二维码的插件，方便手机扫一扫

Microsoft Remote Desktop 微软远程桌面，需要美国区appleid，没有Jump Desctop好用

Jump Desktop 是MAC OS平台上的一款远程控制程序，支持Windows和Mac 双平台，通过邮件关联即可帮助设备自动找到桌面并进行操作

Dukto 跨平台的局域网文件传输工具

Little Snitch 是一款 Mac OS X 上的个人安全软件，能防止应用程序在你不知道的情况下自动访问网络，你可以定义不允许访问。

PeakHour 实时了解您的网络状况，提供您在实时互联网或无线网络的活动即时视图

Net Monitor 实时网络监控工具版，它不需要用户自行设定便可实时的监控电脑中的各个应用程序所占流量

IP Scanner 一款Mac上很实用的扫描局域网内设备的小工具，能够自动扫描出当前局域网内连接的设备，如路由器、iPhone、iPad、打印机，列表显示设备的名称、IP地址、MAC地址等信息，此外还能够为某个设备进行 Ping、WOL、Port Scan等命令

Paw 实用的HTTP/REST服务测试工具

ShadowsocksX

Lantern

Gas Mask Hosts file manager for OS X

lhosts

写作

MacDown open source Markdown editor for OS X

Ulysses 强大的文本编辑器markdown

Byword 文字编辑软件markdown

Marked 优秀的Markdown预览工具
